Overview

MyWorkspace is a new staff frontline tooling platform, with a core set of capabilities for the majority of our servicing needs, and a comprehensive customer landing page that acts as a single navigation point to functionalities and existing applications in place.

Process

Background

Today: Too many windows/too many systems; fragmented experience for staff.
Future and now: MWS is getting live by market and business according to the roadmap. We are building new future-proofing staff features (Initiatives) that can be part of the MWS.

Today's banking systems

Project Context

DXX is one of the functions to be built in MWS. It allows staff to perform an end-to-end foreign exchange journey and make the deal with customer under a confirmed exchange rate.

Design Process

Identify problem

Currently, branch staffs need to input transaction detail in [legacy system name], and check the real-time exchange rate in another system, the rate might vary when the transaction proceeds, which is not promising to the customer.
With the new implementation, branch staff can confirm the rate with the customer and complete the transaction while the rate is still valid.
